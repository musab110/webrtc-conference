# WebRTC Conference Server مع OpenSSL 3.5.3

مشروع خادم مؤتمرات الفيديو مع دعم HTTPS الآمن باستخدام OpenSSL 3.5.3

## الميزات

- 🚀 **خادم HTTPS آمن** مع OpenSSL 3.5.3
- 📹 **مؤتمرات فيديو** متعددة المشاركين
- 💬 **محادثة نصية** فورية
- 🔐 **اتصال آمن** مشفر
- 🌐 **واجهة عربية** سهلة الاستخدام
- 👥 **إدارة الغرف** والمشاركين

## متطلبات النظام

- **Node.js** >= 16.0.0
- **OpenSSL** >= 3.0.0 (مثبت في C:\OpenSSL)
- **متصفح حديث** يدعم WebRTC

## التثبيت والإعداد

### 1. تثبيت التبعيات
```bash
npm install
```

### 2. التحقق من الإعداد
```bash
npm run setup
```

### 3. تشغيل الخادم
```bash
npm start
```

### 4. فتح التطبيق
افتح المتصفح على: `https://localhost:443`

## إنشاء شهادات SSL

إذا كنت بحاجة لإنشاء شهادات SSL جديدة:

```bash
cd cert
openssl genrsa -out key.pem 2048
openssl req -new -key key.pem -out cert.csr -subj "/C=US/ST=State/L=City/O=Organization/CN=localhost"
openssl x509 -req -days 365 -in cert.csr -signkey key.pem -out cert.pem
```

## الاستخدام

1. **إنشاء غرفة**: أدخل اسمك واضغط "إنشاء غرفة"
2. **مشاركة الغرفة**: شارك رابط الغرفة مع الآخرين
3. **الانضمام للغرفة**: أدخل رمز الغرفة واطلب الانضمام
4. **إدارة المشاركين**: المضيف يمكنه قبول/رفض الطلبات

## الأوامر المتاحة

- `npm start` - تشغيل الخادم
- `npm run setup` - فحص الإعدادات
- `npm run test-certs` - اختبار الشهادات

## الإعدادات

يمكن تخصيص الإعدادات من خلال ملف `.env`:

```env
PORT=443
SSL_KEY_PATH=cert/key.pem
SSL_CERT_PATH=cert/cert.pem
```

## استكشاف الأخطاء

