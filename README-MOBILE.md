# دليل الاتصال من الهاتف المحمول 📱

## الطريقة الأولى: استخدام عنوان IP المحلي

### الخطوة 1: معرفة عنوان IP الخاص بالكمبيوتر
```bash
# في Terminal على الكمبيوتر:
ipconfig

# ابحث عن سطر يشبه:
# IPv4 Address. . . . . . . . . . . : 192.168.1.100
# أو
# 192.168.56.1
```

### الخطوة 2: التأكد من أن الخادم يعمل على جميع العناوين
```javascript
// في server.js - تأكد من أن PORT يساوي 3000
const PORT = process.env.PORT || 3000;
```

### الخطوة 3: الوصول من الهاتف
1. **تأكد من أن الهاتف متصل بنفس شبكة WiFi**
2. **افتح المتصفح على الهاتف**
3. **اكتب العنوان:**
   ```
   https://192.168.1.100:3000
   ```
   (استبدل 192.168.8.91 بعنوان IP الفعلي)

### الخطوة 4: تجاوز تحذير الأمان
1. انقر على **"متقدم"**
2. انقر على **"المتابعة إلى [IP]"**

## الطريقة الثانية: استخدام اسم الكمبيوتر

### إذا كان الكمبيوتر يدعى "DESKTOP-ABC":

1. **اكتب في الهاتف:**
   ```
   https://DESKTOP-ABC:3000
   ```

2. **أو استخدم:**
   ```
   https://DESKTOP-ABC.local:3000
   ```

## استكشاف الأخطاء:

### إذا لم يعمل الاتصال:

1. **تأكد من أن الخادم يعمل:**
   ```bash
   npm start
   ```

2. **تأكد من أن المنفذ 3000 متاح:**
   ```bash
   netstat -an | findstr :3000
   ```

3. **تأكد من إيقاف Firewall مؤقتاً:**
   - ابحث عن "Windows Defender Firewall"
   - أضف استثناء للمنفذ 3000

4. **جرب منفذ مختلف:**
   ```javascript
   const PORT = 8080; // بدلاً من 3000
   ```

## للاختبار السريع:

```bash
# على الكمبيوتر - اختبر الاتصال المحلي:
curl -k https://localhost:3000

# يجب أن ترى محتوى HTML
```

## ملاحظات مهمة:

- ✅ **HTTPS مطلوب** - لن يعمل HTTP
- ✅ **المنفذ 3000** - أو أي منفذ تستخدمه
- ✅ **نفس الشبكة** - WiFi نفسها
- ✅ **تجاوز الأمان** - طبيعي للشهادات الذاتية

## بعد الاتصال الناجح:

1. **أدخل اسماً** في حقل "اسمك"
2. **انقر "إنشاء غرفة"** لإنشاء غرفة جديدة
3. **انسخ رابط الغرفة** من المربع
4. **أرسل الرابط للشخص الآخر**
